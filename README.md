# HRlytics: HR Database Analytics System Using SQL

## Overview

HRlytics is a SQL-based HR Database Analytics System developed to manage and analyze employee data. The project stores information related to employees, departments, jobs, locations, countries, regions, and dependents.

Using SQL queries, the system helps generate insights about workforce distribution, salary analysis, department performance, and management hierarchy.



## Objectives

- Manage employee records efficiently
- Maintain relationships between HR tables
- Perform employee and salary analysis
- Generate HR reports using SQL
- Support data-driven decision-making



## Database Tables

The project contains the following tables:

- Regions: Stores geographical regions.
- Countries: Stores country details.
- Locations: Stores office locations.
- Departments: Stores department information.
- Jobs: Stores job titles and salary ranges.
- Employees: Stores employee details.
- Dependents: Stores dependent information.

### Location Analysis

## Key Findings

- Executive departments contain senior management roles.
- Salary levels vary across departments.
- Some departments have more employees than others.
- Certain managers supervise larger teams.
- Employees are distributed across multiple locations.



## Business Insights

- Improves visibility of reporting structures.
